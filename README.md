# Doubly-Linked-List
Implementation of a doubly linked list in Java

The program has the following functionalities:
  1. Insert Head 
  2. Insert Tail 
  3. Insert Before
  4. Insert After
  5. Delete Certain Key
  6. Delete Head
  7. Delete Tail
  8. Display Forwards
  9. Display Backwards
  10. Display Previous
  11. Display Next
  12. Display Head
  13. Display Tail
  14. Search

The doubly linked list implemented does not allow for duplicates and holds integers
